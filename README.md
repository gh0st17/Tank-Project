# Tank-Project
SFML and Box2D based game

# Controls:

- W, S - Moving.
- D - Debug.
- P - Rescue.
- F1 - Select Centauro120.
- F2 - Select Ural.
- F3 - Select VAZ 2107.
- Space - Handbreak.
- LShift - Acceleration.
- RMB - Slowmotion.
- ESC - Exit.
